Block-Based WordPress Theme with Dark Mode, by Seed Webs.

== Description ==

Bai is block-based WordPress theme with dark mode that supports full-site editing. It comes with a set of minimal templates and design settings that can be manipulated through Global Styles.

== Privacy ==
Bai uses LocalStorage to save the setting when Dark Mode support is turned on or off.
LocalStorage is necessary for the setting to work and is only used when a user clicks on the Dark Mode button.
No data is saved in the database or transferred.
